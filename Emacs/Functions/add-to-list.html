add-to-list

add-to-list는 subr.el 에 정의되어 있으며 컴파일된 Lisp 함수이다. 

(add-to-list LIST-VAR ELEMENT &optional APPEND COMPARE-FN)

ELEMENT를 LIST-VAR의 값(어퍼스트로피를 의미하는 듯)에 추가한다. 만일 list에 같은 ELEMENT가 있다면 추가되지 않는다. ELEMENT와 같음을 판별하기 위해 'equal'을 사용하거나 COMPARE-FN을 지정해서 사용한다. ELEMENT가 추가되는 경우 기본적으로는 list의 시작 부분에 추가된다. 끝 부분에 추가하기 위해서는 APPEND 인자에 nil이 아닌 값을 넣어 주면 된다.

리턴값은 업데이트된 LIST-VAR 이다.

만일 특정 패키지가 로드되지 않은 상태에서는 정의되지 않는 변수에 'add-to-list'를 사용하고 싶다면 패키지 로딩 후에 실행되는 hook 함수에서 'add-to-list'를 부르면 된다. 'eval-after-load' 문서에서 이러한 방법을 소개한다. 매이저 모드 hook과 같은 경우에서 이런 방식을 사용한다.
